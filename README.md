BMI Calculator 🧮

A simple Python BMI (Body Mass Index) Calculator made while learning Python from the 100 Days of Code - Python Bootcamp course.

📌 Features

- Takes weight and height as input
- Calculates BMI using the BMI formula
- Shows whether the person is:
  - Underweight
  - Normal weight
  - Overweight

🚀 Technologies Used

- Python

📖 BMI Formula

BMI is calculated using:

bmi = weight / (height ** 2)

▶️ How to Run

1. Install Python on your computer
2. Download the project
3. Open the file in PyCharm or VS Code
4. Run the Python file
